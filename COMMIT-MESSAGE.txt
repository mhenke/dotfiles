Complete Linux Mint + i3 migration setup

Added comprehensive dotfiles management system for easy migration to new machines.

## New Features:
- One-command automated setup via bootstrap.sh
- GNU Stow integration for clean symlink management
- Complete package backup system (2,922 packages documented)
- Automated installation scripts for all components
- Comprehensive documentation (README, MIGRATION guide, QUICKREF)

## Added Scripts:
- bootstrap.sh - Main automated setup script
- verify-setup.sh - Verification script
- scripts/backup-system.sh - Generate package lists
- scripts/install-packages.sh - Install system packages
- scripts/install-dev-tools.sh - Install Node.js, Ruby, Python, AWS, etc.
- scripts/install-apps.sh - Install VSCode, Bitwarden, Discord, Kodi
- scripts/setup-stow.sh - Symlink all configs with Stow
